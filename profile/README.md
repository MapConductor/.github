# Map Conductor - Unified Map SDK API

![flyer](https://github.com/user-attachments/assets/d2410507-de0f-4b77-a761-83ad6a157ec5)

MapConductor is an abstraction layer over multiple Map SDKs.
It means you can manipulate any SDKs supported by MapConductor with one common syntax code.

We currently support:

[Android]
- Google Maps Android API version 2.x
- Mapbox Native for Android version 11.x
- MapLibre Native for Android version 12.x
- HERE SDK version 4.x
- ArcGIS SDK for Kotlin version 200.x

[iOS]
- MapKit
- Google Maps iOS API version 10.x
- Mapbox Native for iOS version 11.x
- MapLibre Native for iOS version 12.x
- ArcGIS SDK for Kotlin version 200.x

Get started at https://mapconductor.com
